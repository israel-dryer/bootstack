# Capabilities (Widget Interface)

These pages document **capability slices** of the bootstack widget interface.
Capabilities are implemented via internal modules/mixins, but the behavior they provide is **public** through widgets.

> Note: You typically won’t instantiate these mixins directly. Use the normal widget classes; these pages describe
> the behavior and methods/options they provide.
